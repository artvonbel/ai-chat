# 🤖 AI Бот-Бриф (Техническое Задание)

[![Демо](https://img.shields.io/badge/демо-онлайн-brightgreen)](https://ai-chat-artie.netlify.app/)
[![MIT License](https://img.shields.io/badge/license-MIT-blue)](./LICENSE)

**AI-Бриф** — это интеллектуальный чат-бот, который помогает клиентам и фрилансерам быстро составить структурированное Техническое Задание (ТЗ) для веб-проектов. Бот задаёт наводящие вопросы, уточняет детали и выдаёт готовый документ, который можно скачать в формате `.txt` или скопировать в буфер обмена.

Проект создан для портфолио и демонстрирует навыки интеграции нейросетей, работы с API и разработки полноценного веб-интерфейса.

## ✨ Особенности

- ✅ **Генерация ТЗ** — бот анализирует ответы пользователя и формирует чёткий структурированный документ.
- ✅ **Готовые примеры** — четыре кнопки с типовыми запросами для быстрого старта.
- ✅ **Экспорт результата** — скачивание полного диалога или только финального ТЗ, копирование в буфер.
- ✅ **Тёмная/светлая тема** — переключается по клику, выбор сохраняется в браузере.
- ✅ **Сохранение истории** — диалог не пропадает после обновления страницы (хранится в `localStorage`).
- ✅ **Адаптивный дизайн** — корректно отображается на всех устройствах (телефоны, планшеты, десктопы).
- ✅ **Индикатор печати** — пользователь видит, что бот обрабатывает запрос.
- ✅ **Обработка ошибок** — понятные сообщения при сбоях сервера или тайм-ауте.

## 🛠️ Технологии

### Бэкенд
- **Node.js** + **Express** — серверная часть.
- **LangChain** — фреймворк для работы с LLM.
- **OpenRouter** — доступ к бесплатным моделям (GPT-3.5-подобные, Llama 3.3, Gemma и др.).
- **CORS** — для кросс-доменных запросов.

### Фронтенд
- **HTML5** + **CSS3** — семантическая вёрстка, CSS-переменные для тем.
- **JavaScript (ES6+)** — вся логика чата, работа с `localStorage`, `fetch API`, экспорт файлов.

### Деплой
- **Render** — хостинг бэкенда.
- **Netlify** — хостинг фронтенда.

## 🧑‍💻 Как запустить локально

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/artvonbel/ai-chat.git
cd ai-chat

2. Установите зависимости бэкенда
bash
npm install
3. Создайте файл .env в корне проекта
Вставьте в него ваш API-ключ от OpenRouter (получить можно на openrouter.ai):

env
OPENAI_API_KEY=ваш_ключ_здесь
4. Запустите сервер
bash
npm start
Сервер запустится на порту 5000 (или на порту, указанном в переменной PORT).

5. Откройте фронтенд
Просто откройте файл frontend/index.html в браузере или

Используйте Live Server в VS Code (рекомендуется для горячей перезагрузки).

В файле frontend/app.js убедитесь, что переменная API_URL указывает на адрес вашего бэкенда. По умолчанию: http://localhost:5000/api/chat.
