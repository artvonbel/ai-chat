# 🤖 AI Бот-Бриф (Техническое Задание)

Этот проект — умный чат-бот для фрилансеров и клиентов. Он помогает составить структурированное Техническое Задание (ТЗ) для веб-проектов, задавая наводящие вопросы.

**Демо:** *[Ссылка на твой будущий хостинг, если зальешь]*

## 🚀 Технологии

- **Backend:** Node.js + Express
- **AI/LLM:** OpenRouter (бесплатные модели, например `openrouter/free`)
- **Интеграция:** LangChain.js
- **Frontend:** Vanilla HTML + CSS (чистый JavaScript)

## 📂 Структура проекта

- `/backend` — серверная часть (API)
- `/frontend` — интерфейс чата
- `brief_instructions.txt` — база знаний бота (инструкция по составлению ТЗ)
